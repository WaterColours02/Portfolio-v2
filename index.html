open-ferment

Javascript - Node - Mail Chimp Api - HTML - CSS

The Vancouver Wine Festival Demo Website is an unreleased project created in collaboration with a client and designer. It showcases the allure of the wine festival in Vancouver through a captivating splash page. The website includes a Mailchimp email sign-up integration, allowing visitors to stay informed about the festival. The site utilizes a full stack approach, with a static front end and a node.js server app that communicates with the Mailchimp API. Although not publicly released, this project demonstrates expertise in creating immersive web experiences that seamlessly blend design and functionality for wine festivals.
